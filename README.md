
# Game memory

Esse jogo teve como foco adaptar o projeto desenvolvido pelo [Manual do Dev](https://youtu.be/NV88N1r2Qkg) feito em HTML, CSS e JS para a biblioteca [PixiJS](https://pixijs.io/).

## 🛠 Tecnologias/bibliotecas

- PixiJS v6.4.2
    - pixi-sound v4.2.0
    - TweenMax.min v2.1.3
